How Marks Are Calculated

Each question has 7 options with multiple correct answers, and is worth 2 marks. Marks are awarded and penalised per correct option as follows:

+k
Each correct option chosen
−k
Each wrong option chosen
−k
Each correct option missed

Where k = 2 ÷ (no. of correct options). This normalises scores across questions with different numbers of correct answers. Minimum per question = 0 (no negative carry-over).

  • Select all answers you believe are correct — partial selection is penalised.
  • You will see your score and a detailed rationale immediately on submission.
  • Questions test analytical reasoning, not rote recall.
